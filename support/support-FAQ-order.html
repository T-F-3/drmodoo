Q
주문/결제 입금자명과 입금 은행을 변경하고 싶어요.

주문 시 기재해주신 입금 정보와 다를 경우 입금 확인이 어렵습니다. 이미 주문을 하신 경우 입금 정보 변경이 불가하오니 변경을 원하실 경우 번거로우시겠지만 다시 주문해주시기 바랍니다.

Q
주문/결제 입금했는데 '입금 확인 중'이라고 나와요.

수시로 입금 확인이 진행되는 것이 아니라 영업 시간 내 정해진 시간마다 일괄적으로 입금 확인이 진행되므로 조금만 기다려주시면 입금 확인이 됩니다. 혹, 입금 후 24시간이 지났음에도 ‘입금 확인 중’이라고 나오는 경우 고객센터로 연락주시면 신속히 확인해드리도록 하겠습니다.

* 아래의 경우 반드시 고객센터로 연락주셔야 입금 확인이 가능합니다.

1) 주문 확인이 어려운 경우
- 주문시 작성한 입금자명과 실제 입금자명이 다른 경우
- 입금 해야될 금액과 실제 입금한 금액이 다른 경우
- 입금 기한이 지난 후 입금한 경우 (입금 기한 : 주문일로부터 7일 이내)

Q
주문/결제 결제 후 취소는 어떻게 하나요?

주문 내역에서 직접 취소 신청을 하시거나 고객센터로 연락주시면 결제 취소를 도와드리겠습니다.
